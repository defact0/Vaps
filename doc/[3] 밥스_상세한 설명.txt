화면에 보이는 내용을 정의


[ 첫 페이지 ]

        [1] 메뉴 

                1. 상품 목록 보기

                2. 명예 전당

                3. 회원 가입

                4. 사이트 소개


        [2] 메뉴 외 기본 페이지에서 보여지는 정보

                1. 로그인할 수 있는 영역!

                2. 랜덤 으로 출력 되는 추천 메뉴 정보

                3. 사이트 회원가입을 유도하는 홍보성 공간


[ 회원가입 ]

        [1] 메뉴외 기본페이지에서 보여지는 정보

                메뉴는 없고 회원 정보 입력과 첫페이지로 돌아가는 기능 밖에 없다.        

                - 관리자로 회원가입을 할 수 없다.(DB에서 직접 가입해야 쓸 수 있다)

                - 모든 항목을 입력해야 INSERT 명령을 수행할 수 있도록 웹에서 검사하는 기능

                - 아이디와 닉네임 중복을 검사하는 기능을 추가

                - 회원가입시, 비밀번호 두번 입력


[ 구매자 ]

        [1] 메뉴

                1. 상품 목록 보기

                        - 완성품 도시락을 주문할 수 있는 메뉴들이 나오는 페이지

                        - 형식은 의류 쇼핑몰 처럼 이미지가 크게 나오는 형태로 출력

                        - 선택하면 장바구니로 담아지게 되고 아직 DB에는 저장이 안되는 형태로 작성


                2. 맞춤 도시락

                        - 커스텀 형태의 도시락 주문

                        - 도시락 각 구역을 지정해 미리 준비해 놓은 재료를 골라서 주문하게 만든다.


                3. 장바구니

                        - 세션이 끊어지기 전까지 구매한 도시락들이 임시저장되어 보여지는 페이지이다.

                        - 이곳에서 추가적으로 곱배기 혹은 추가 물품(컵라면+김치) 같은 것들을 구입 가능하게 제작


                        - ★ 결제를 누르면 영수증 같은 화면 추가!!!


                4. 주문관리

                        - 포인트 확인 (일괄적으로 10번 구입하면 1번은 공짜로 구입 가능하게 제작)

                        - 그간 도시락 주문 현황을 확인가능한 페이지 (구입 횟수, 평균 구입가격, MAX, MIN 가격 등등...)


                5. 회원정보

                        - 회원 가입시 입력한 정보를 수정할수 있다.

                        - 아이디는 변경이 불가능

                        - 패스워드는 변경이 가능하나, 패스워드 입력확인을 하는 기능을 추가했으면 한다.

                          (패스워드를 두번입력하고 그게 일치하면 성공!)


                6. 고객센터

                        - 게시판 형식의 QA이다.

                        - 답글이 가능한 게시판으로 만들어야 하고, 게시글 등록 시에는 비밀번호가 필요 없게 만들어야 한다.

                        - 게시글 삭제는 작성자만 혹은 운영자가 삭제 할 수 있어야 한다


                7. 명예전당(옵션)

                        - 추천을 많이 받은 도시락(커스텀 도시락XXX)

                        - 주간 많이 구매한 사람 top5

                        - 월간 도시락 왕 top5

                        - 다양한 기록들을 열람 할 수 있다.(만들 수 있다면은...)


        [2] 메뉴외 기본페이지에서 보여지는 정보

                1. 포인트 정보(간단)

                2. QA게시판 글 5개 출력

                3. 추천메뉴(랜덤)


[ 관리자 ]

        [1] 메뉴

                1. 주문관리 ( 각회원들이 주문한 것을 최근 순서대로 보여주기)

                2. 상품목록관리 ( 판매하는 도시락을 추가 및 수정, 이미지 혹은 내용 수정등등)

                3. 매출관리 ( 일,주,월,년도로 통계 ) 

                4. 회원관리

                        - 비밀번호는 암호화, 강제적으로 비밀번호를 초기화시켜주는 용도로....(1111번으로 바꾸던지..)


                5. 재고관리

                6. 고객센터


        [2] 메뉴외 기본페이지에서 보여지는 정보

                1. QA게시판 글 5개 출력

                2. 오늘의 매출 정보

                3. 최근 가입한 회원 (5개이상?)

                4. 재고 정보(간단하게)
