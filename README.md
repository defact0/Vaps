Vaps Project!
====

밥스 프로젝트는 **Team Project** 으로 진행됩니다.
밥스는 온라인 도시락 판매관리 Web site 입니다.




###밥스(Vaps) 라는 이름의 유래

>밥스는 빕스(Vips)라는 브랜드명을 패러디 했습니다.
